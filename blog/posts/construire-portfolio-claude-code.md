# Comment j'ai construit mon portfolio avec Claude Code

*Publié le 7 février 2026*

## Introduction

Créer un portfolio de développeur est un rite de passage. Mais cette fois, j'ai décidé de le faire différemment : en utilisant **Claude Code** comme assistant de programmation.

## Pourquoi Claude Code ?

L'idée était simple : apprendre en construisant, tout en ayant un mentor IA disponible 24/7. Voici ce que j'ai découvert.

### Les avantages

- **Feedback instantané** : Chaque ligne de code peut être revue
- **Apprentissage contextuel** : Les explications sont adaptées au projet
- **Productivité** : Moins de temps à chercher sur Stack Overflow

## Le code en action

Voici un exemple de fonction JavaScript que nous avons créée ensemble :

```javascript
function toggleTheme() {
    var currentTheme = root.classList.contains('light-mode') ? 'light' : 'dark';
    var newTheme = currentTheme === 'light' ? 'dark' : 'light';
    setTheme(newTheme);
}
```

Et du CSS avec des variables :

```css
:root {
    --color-bg: #0a0a0a;
    --color-accent: #64ffda;
}
```

## Conclusion

Construire ce portfolio m'a appris autant sur le développement web que sur la collaboration avec l'IA. Le résultat : un site performant, accessible, et entièrement personnalisé.
